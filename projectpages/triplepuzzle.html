GAMEPLAY

"triplepuzzle." was a single Level Design entry for the ongoing Source Engine anthology project "Map Labs" and its "Three Rooms" challenge, planned, started and finished in about 4 days.

ROLES & RESPONSIBILITIES

Since "triplepuzzle." was a solo project, I had the roles and responsibilities of both planning and implementing the Level Design

KEY SOFTWARE

Source SDK

Adobe Photoshop

CONTRIBUTIONS

  • Rapid protyping of each of the three rooms, working on each separately.
  • Created custom textures using Photoshop.
  • Made usage of the Source Engine's Input/Output system to script and implement multiple endings.
  • Ported assets from other Source Engine title, Counter-Strike: Source.
